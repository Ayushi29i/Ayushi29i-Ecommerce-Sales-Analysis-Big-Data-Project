 # 🛒 E-COMMERCE SALES ANALYSIS USING HADOOP
 ###  A complete Big Data pipeline to process and analyze e-commerce sales data using Hadoop (HDFS, MapReduce, Hive)

 #  Big Data Fundamentals Project  
- Implemented using Hadoop Ecosystem (HDFS, MapReduce, Hive)

 # 📌 PROJECT OVERVIEW
- This project demonstrates a complete Big Data processing pipeline to analyze e-commerce sales data using Hadoop.

# 🎯 OBJECTIVES
✔ Identify top-selling products  
✔ Calculate total sales  
✔ Analyze sales trends  
✔ Find peak sales dates  

# 🏗️ ARCHITECTURE
### 🔄 Data Flow:
**Raw Data → HDFS → MapReduce → Hive → Insights**

# ⚙️ TECHNOLOGIES & TOOLS USED
- Hadoop (HDFS, MapReduce)  
- Hive  
- Python  
- Linux  
- Cloudera  

# 📂 DATASET INFORMATION
- File: `ecommerce.csv`  
- Fields: OrderID, Product, Category, Price, Quantity, Date  


# 🔄 WORKFLOW

## - Step 1: Data Storage (HDFS)
Dataset uploaded into HDFS  

## - Step 2: Processing (MapReduce)
- Mapper → (product, 1)  
- Reducer → total count  

## - Step 3: Analysis (Hive)
- GROUP BY  
- ORDER BY  
- SUM
